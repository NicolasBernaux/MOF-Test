##   Installation

To install this project :

- Update gulp-cli `sudo npm i -g gulp-cli`

`$ cd your repository`

`$ git clone this project git's url`

`$ cd folder-name`

`$ npm install`

## Edit
- Simple task `gulp`
- Watch and server task `gulp run`
